## General info
This project is blackjack a card game that pits player versus dealer.
	
## Technologies
Project is created with:
* React: 17.0.1
* Typescript: 4.1.2
* Sass: 4.14.1
* Bootstrap: 4.6.0
	
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```

## GitHub Pages Setup

To put this project on GH, install it using npm:

```
$ npm install
$ npm run deploy
```
